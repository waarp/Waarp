Waarp's Family

We present here the "Commons" modules in Waarp project.

 

  1. Waarp Digest: This library takes into account CRC/MD5/SHA in an optimized way.
  2. Waarp Common: Main common library shared with all projects in GoldenGate.
  3. Waarp LocalExec Daemon: used to execute on a simple external JVM the external command
  4. Waarp XmlEditor: based on XML Editor from F Golubov
  5. Waarp Password: used to generate DES key and passwords in GGP format
  6. Waarp SNMP: used to implement the SNMP Agent support for all Waarp project based on SNMP4J 
  7. Waarp Gateway Kernel: used by Gateway FTP and Gateway HTTP for shared methods (database logs, actions, http shared package)

  8. Waarp Thrift: used by Waarp R66 to provide a Thrift interface for other languages than Java

  9. Waarp FtpClient: a simple wrapper around 2 FTP client library (Apache and FTP4J) used by Waarp R66

  10. Waarp Vitam : an integration for secure, production ready of Waarp and Vitam to ensure SIP and DIP transfers for digital archiving

  11. Waarp HTTP : Waarp Http prototype support for end users through Browser

    Waarp Vitam : Waarp Vitam allows to use Waarp and Vitam project to send or receive archives (see Vitam Project).
